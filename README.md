# MOJO

## Built with
* Keras

## Getting Started
To use Keras, will need to have the TensorFlow package installed. [See detailed instructions](https://www.tensorflow.org/install).
  
Once TensorFlow is installed, import Keras via:
```
from tensorflow import keras
```
